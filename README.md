# 🛍️ E-Commerce Web Application

A modern and fully responsive e-commerce web application built with **Next.js**, designed to deliver a fast, smooth, and user-friendly shopping experience. The app allows users to explore products, browse categories, and manage their shopping cart with ease, powered by a real REST API.

---

## 🚀 Features

* 🔐 User-friendly interface with responsive design
* 🛒 Add, remove, and manage shopping cart items
* 📦 Browse products and filter by categories
* ⚡ Fast performance with optimized rendering using Next.js
* 🔄 Dynamic data fetching from a real REST API
* 🎯 Clean and modern UI for better user experience

---

## 🛠️ Tech Stack

* **Frontend:** Next.js, React
* **Styling:** Html / CSS / Bootstrap / Tailwind / Next.js
* **API:** REST API
* **Deployment:** Vercel

---

## 📦 Getting Started

First, clone the repository:

```bash
git clone https://github.com/olatarek01/E-commerce
```

Then install dependencies:

```bash
npm install
```

Run the development server:

```bash
npm run dev
```

Open your browser and go to:

```
http://localhost:3000
```
